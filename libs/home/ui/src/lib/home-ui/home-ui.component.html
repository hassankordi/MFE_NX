Micro Frontends
Micro Frontends Means Split One Large Application Into Smaller Applications Read more
Module Federation
Module Federation is a Webpack feature that allows different apps (MFEs) to share code dynamically at runtime.
Monorepo Integratoin (NX)
A Monorepo is a single repository that contains multiple applications and libraries. Nx is a powerful tool for managing Monorepos in Angular, React, Node.js, and more.
